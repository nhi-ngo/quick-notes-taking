## [**Quick-Notes-Taking**](https://quick-notes-taking.netlify.app/)

# **Tech**
- React
- MongoDB
- Express
- Node

# **Features**
- Create/Edit/Update/Delete a note
- Upload an image from local computer
- React to a note with a like icon
